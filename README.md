# View Planning for Repeated Plant Monitoring

This is a repository for supporting our paper "[Efficient View Planning Guided by Previous-Session Reconstruction for Repeated Plant Monitoring](https://arxiv.org/abs/2510.07028)" by S. Pan, L. Lobefaro, M. Taherkhani, X. Huang, R. Menon, C. Stachniss, and M. Bennewitz.  

An earlier version "[Temporal-Prior-Guided View Planning for Periodic 3D Plant Reconstruction](https://activep-ws.github.io/accepted_papers/5_Temporal_Prior_Guided_View_P.pdf)" has been presented in Active Pecpertion Workshop at IROS 2025.

## single_plant

This folder contains our single-plant experimental codes.

## greenhouse_row

This folder contains our greenhouse-row experimental codes.
